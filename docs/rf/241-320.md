# Módulo 7 — Pedidos (OMS - Order Management System)

## Objetivo

O módulo de Pedidos é responsável por gerenciar todo o ciclo de vida de um pedido, desde sua criação até sua conclusão, cancelamento ou devolução.

O OMS atua como orquestrador dos demais módulos do sistema, integrando clientes, catálogo, estoque, pagamentos, logística, promoções e notificações.

---

# Criação do Pedido

## RF241 — Criação de Pedido

### Objetivo

Permitir criar um novo pedido.

### Descrição

O sistema deverá permitir criar pedidos contendo um ou mais produtos.

### Regras de Negócio

- O cliente deverá estar ativo.
- Todos os produtos deverão estar ativos.
- O pedido deverá possuir ao menos um item.

### Dependências

RF031
RF075

### Critérios de Aceite

- O pedido é criado.
- O status inicial é **Rascunho**.
