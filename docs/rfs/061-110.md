# Módulo 4 — Catálogo

## Objetivo

O módulo de Catálogo é responsável pelo gerenciamento das informações comerciais dos produtos disponibilizados pelo sistema.

Ele centraliza produtos, categorias, marcas, fornecedores, atributos, variações, preços e demais informações necessárias para comercialização.

---

# Categorias

## RF061 — Cadastro de Categoria

### Descrição

O sistema deverá permitir cadastrar categorias de produtos.

### Regras

- O nome deverá ser único dentro da mesma categoria pai.
- Categorias poderão possuir categoria pai.

### Critérios de Aceite

- Categoria cadastrada.
- Auditoria registrada.

---

## RF062 — Atualização de Categoria

### Descrição

Permitir alterar informações da categoria.

---

## RF063 — Exclusão Lógica de Categoria

### Descrição

Permitir desativar categorias.

### Regras

Não será permitido excluir categorias que possuam produtos vinculados.

---

## RF064 — Hierarquia de Categorias

### Descrição

O sistema deverá permitir categorias em múltiplos níveis.

Exemplo

Alimentos

→ Bebidas

→ Refrigerantes

→ Zero Açúcar
