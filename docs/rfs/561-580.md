# Indicadores e Métricas do Dashboard

## RF561 — Indicadores de Vendas

### Objetivo

Disponibilizar indicadores relacionados às vendas realizadas pelo sistema.

### Descrição

O sistema deverá consolidar informações referentes ao desempenho das vendas, permitindo acompanhar métricas estratégicas e operacionais.

### Regras de Negócio

- Os indicadores deverão considerar apenas pedidos válidos.
- Os valores deverão respeitar os filtros aplicados.
- Os indicadores deverão ser atualizados automaticamente.

### Dependências

- RF241

### Critérios de Aceite

- Os indicadores são apresentados corretamente.
- Os valores permanecem consistentes.

---
